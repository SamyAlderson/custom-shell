# custom-shell

## Description

Implémentation d'une shell personnalisée en C.

## Contexte

Ce projet implémente une shell personnalisée en C, avec des fonctionnalités telles que la gestion de l'historique de commandes, l'autocomplétion et des commandes personnalisées.

## Fichiers

### src/shell.c

* Fichier principal de la shell

### src/history.c

* Gestion de l'historique de commandes

### src/autocomplete.c

* Fonctionnalité d'autocomplétion

### src/commands.c

* Implémentation de commandes personnalisées

### src/parser.c

* Analyseur de commande

### Makefile

* Fichier de compilation

### README.md

* Documentation du projet

## Fonctionnalités

* Gestion de l'historique de commandes
* Fonctionnalité d'autocomplétion
* Implémentation de commandes personnalisées

## Dépendances

* bash
* readline

## Langage

C

## Domaine

Utils

## Installation

Pour compiler le projet, utilisez la commande suivante :

```bash
make
```

Pour exécuter la shell, utilisez la commande suivante :

```bash
./custom-shell
```
